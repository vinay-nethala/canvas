# Sketchpad

A small React + Vite drawing app (pen, eraser, line, rectangle) with save/export features.

## Requirements

- Node.js 18+ recommended

## Installation

Install dependencies:

```bash
npm install
```

## Development

Start the dev server:

```bash
npm run dev
```

Open http://localhost:5173 in your browser.

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```
